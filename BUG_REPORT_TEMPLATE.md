## 严重程度定义参考

| 等级 | 定义 | 示例 |
|------|------|------|
| **P0** | 阻塞性缺陷，核心功能完全不可用，无 workaround | Save Summary 和 Inject 均无法使用 |
| **P1** | 严重缺陷，主要功能受影响，但有临时解决方案 | Save Summary 偶发失败，重试可成功 |
| **P2** | 一般缺陷，次要功能问题或边界场景 | 特定页面 Inject 失败 |
| **P3** | 轻微缺陷，UI/UX 优化建议或极低频问题 | 列表排序显示不一致 |

## 提交指南

1. 复制本模板到新文件
2. 填写所有适用字段（不适用的可标注 N/A）
3. 命名格式：`BUG-CC-YYYYMMDD-XXX.md`
4. 提交到项目 issue 追踪系统或发送给开发团队
