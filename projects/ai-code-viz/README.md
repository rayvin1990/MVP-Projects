---

## 🎯 项目目标

解决 Vibe Coding 后遗症状：
- AI 生成代码后失去代码库心理模型
- 无法可视化文件/函数连接关系
- 重构时害怕破坏"魔法"

---

## 📋 MVP 功能范围（Phase 1）

### 核心功能（2-3 天）

1. **代码依赖图可视化**
   - [ ] 文件间调用关系
   - [ ] 函数依赖链
   - [ ] 交互式图形界面

2. **数据流追踪**
   - [ ] 变量传递路径
   - [ ] 组件间数据流动

3. **快速导航**
   - [ ] 点击节点跳转到代码
   - [ ] 搜索功能

## 🏗️ 技术栈

| 层级 | 技术 |
|------|------|
| **前端框架** | React 18 + Vite |
| **可视化库** | React Flow / Cytoscape.js |
| **样式** | TailwindCSS 4 |
| **部署** | Vercel |
| **语言分析** | Tree-sitter (WASM) |
