## 🔄 降级策略

### 步骤 1: 痛点挖掘

**优先级顺序:**
```
1. Reddit (国际用户痛点)
   ↓ 不可用
2. 知乎 + 小红书 (国内用户痛点)
   ↓ 不可用
3. 用户提供数据源
   ↓ 不可用
4. 基于训练数据模拟
```

### 步骤 2: SEO 关键词

**优先级顺序:**
```
1. Google Trends (全球数据)
   ↓ 不可用
2. 百度指数 + 微信指数 (国内数据)
   ↓ 不可用
3. 用户提供关键词
   ↓ 不可用
4. 基于常见关键词推荐
```

### 步骤 9: 部署上线

**优先级顺序:**
```
1. GitHub Pages + Vercel (推荐)
   ↓ 不可用
2. Gitee Pages
   ↓ 不可用
3. 本地运行
   ↓ 不可用
4. 打包 ZIP 交付
```

```json
{
  "researcher": {
    "data_sources": {
      "international": ["Reddit", "Twitter", "Product Hunt"],
      "domestic": ["知乎", "小红书", "微博", "豆瓣"],
      "seo_tools": ["Google Trends", "百度指数", "微信指数"]

    "fallback_strategy": {

      "if_google_blocked": "使用百度指数/微信指数替代",

    }
  }
}
```

## 🔧 切换数据源

### 手动切换

在任务指令中指定：

```json
{
  "step": 1,
  "name": "痛点挖掘",
  "agent": "researcher",
  "data_source": "domestic",  // 或 "international"
  "platforms": ["知乎", "小红书"]
}
```

### 自动检测

系统自动检测网络环境：

```python
def check_network():
    if can_access("reddit.com"):
        return "international"
    elif can_access("zhihu.com"):
        return "domestic"
    else:
        return "offline"
```
