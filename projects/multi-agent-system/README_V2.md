## 🔄 9 步军事化执行流程

### 流程总览

```
┌─────────────────────────────────────────────────────────────┐
│  1️⃣ Reddit 挖痛点     →  需求&SEO Agent                    │
│  2️⃣ SEO 关键词提取    →  需求&SEO Agent                    │
│  3️⃣ 产品极简设计      →  产品 Agent                         │
│  4️⃣ 可行性强制校验    →  Commander-X                        │
│  5️⃣ 指挥官分任务      →  Commander-X                        │
│  6️⃣ 开发 Agent 生成代码 →  开发 Agent                        │
│  7️⃣ 调试 Agent 质检   →  调试 Agent                         │
│  8️⃣ 尖叫度终审        →  产品 Agent + Commander-X           │
│  9️⃣ 部署上线          →  执行 Agent                         │
└─────────────────────────────────────────────────────────────┘
```

### 步骤详解

#### 步骤 1: Reddit 挖痛点
**执行:** 需求&SEO Agent  
**标准:** 高赞刚需、垂直细分、纯前端可做、低维护  
**输出:** 痛点分析报告

#### 步骤 2: SEO 关键词提取
**执行:** 需求&SEO Agent  
**标准:** 核心词 + 长尾词 + 标题描述，同步产出  
**输出:** 关键词列表

#### 步骤 3: 产品极简设计
**执行:** 产品 Agent  
**标准:** 单核心功能 + 原型，内嵌 SEO  
**输出:** PRD 文档

#### 步骤 4: 可行性强制校验
**执行:** Commander-X  
**标准:** 纯前端、无自建后端、免费 API、1 天可上线  
**输出:** 可行性报告

#### 步骤 5: 指挥官分任务
**执行:** Commander-X  
**标准:** 拆解任务、明确 Agent 分工  
**输出:** 任务分配清单

#### 步骤 6: 开发 Agent 生成代码
**执行:** 开发 Agent  
**标准:** 轻量纯前端、可直接运行  
**输出:** 源代码（HTML/CSS/JS）

#### 步骤 7: 调试 Agent 质检
**执行:** 调试 Agent  
**标准:** 逻辑无误 + 界面流畅 + 无报错 + 移动端适配  
**输出:** 测试报告

#### 步骤 8: 尖叫度终审
**执行:** 产品 Agent + Commander-X  
**标准:** 3 秒懂、一步操作、有尖叫点  
**输出:** 尖叫度评分

#### 步骤 9: 部署上线
**执行:** 执行 Agent  
**标准:** 阿里云打包 ZIP→本地→GitHub→发布  
**输出:** 上线 URL

## 📋 核心原则

### 三大铁律（不可违背）

| 原则 | 描述 | 违规处理 |
|------|------|----------|
| **绝对禁令** | 严禁亲自动手执行具体任务 | 立即停止，重新指派 |
| **职能分离** | 严格遵守关注点分离 | 不能越界指挥 |
| **目标导向** | 确保最终目标达成 | 不关注琐碎细节 |

### 自我检查机制

每次行动前问自己：
1. "我是在指派工作，还是我自己在干活？"
2. "任务是否分配给了正确的 Agent？"
3. "是否违反了绝对禁令？"

## 🔍 自我审计

### 心跳检查

```python
# 每 10 分钟自动执行
def heartbeat_audit(self):
    """定期自我审计"""
    recent_actions = self.get_recent_actions()
    
    for action in recent_actions:
        if '违反绝对禁令' in action:
            self.report_violation(action)
            self.correct_action()
    
    self.log("审计完成：无违规")
```

### 权限审查

```python
# 启动时自动执行
def check_permissions(self):
    """检查并禁用执行权限"""
    forbidden_tools = ['code_interpreter', 'file_write', 'shell_exec']
    
    for tool in forbidden_tools:
        if self.has_tool(tool):
            self.disable_tool(tool)
            self.log(f"已禁用：{tool}")
```
