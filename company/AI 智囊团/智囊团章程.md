## 🏛️ 智囊团定位

AI 智囊团是公司的**战略顾问委员会**，由 6 家主流 AI 模型组成，为主任提供多元视角的决策建议。

**核心原则：**
- 顾问角色，无最终决策权
- 独立分析，避免群体思维
- 记忆延续，形成连贯思维
- 免费额度优先，成本控制

### 节省策略

2. **记忆压缩** - 摘要式记忆，节省 token
3. **按需注入** - 只注入相关议题记忆
4. **Nia 预处理** - 简单分析 Nia 完成，复杂问题再咨询

## 📊 决策权重计算

**共识决策：**
- ≥4 模型支持 → 强建议
- 3 模型支持 → 中等建议
- ≤2 模型支持 → 谨慎建议

**权重计算：**
```
建议得分 = Σ(模型权重 × 支持程度)

支持程度：
- 强烈推荐 = 1.0
- 推荐 = 0.7
- 中立 = 0.5
- 谨慎 = 0.3
- 反对 = 0.0

得分 ≥0.7 → 强烈推荐
得分 0.5-0.7 → 推荐
得分 0.3-0.5 → 谨慎
得分 <0.3 → 不推荐
```

## 🧠 记忆系统

### 共享记忆区结构

```
company/共享记忆/
├── 公司背景.md          # 使命/愿景/价值观/组织架构
├── 决策历史/            # 历次咨询归档
│   └── YYYY-MM-DD-议题名称.md
├── 项目进展/            # 进行中项目状态
│   └── 项目名称.md
└── 模型表现档案/        # 各模型历史表现
    ├── ChatGPT.md
    ├── Gemini.md
    └── ...
```

### 记忆注入规范

每次咨询前，Nia 负责注入：
1. **公司背景** - 核心信息（200 字内）
2. **相关决策历史** - 与议题相关的历史决策
3. **项目进展** - 相关项目当前状态
4. **模型角色** - 该模型在智囊团中的定位

## 📈 模型表现评估

### 评估维度

| 维度 | 说明 | 权重 |
|------|------|------|
| 建议质量 | 建议是否被采纳 | 40% |
| 决策结果 | 采纳后效果如何 | 30% |
| 独特视角 | 是否提供独特见解 | 20% |
| 响应速度 | 是否在额度内及时回复 | 10% |

### 评估频率

- **月度评估** - 调整权重和角色
- **季度回顾** - 考虑新增/替换模型

## 📞 沟通机制

### 咨询发起

- **主任发起** - 直接告诉 Nia
- **Nia 建议** - 心跳检查发现需要决策的事项

### 响应时间

- **紧急事项** - 即时咨询（2 小时内）
- **常规事项** - 每日批量处理（18:00 前）
- **复杂事项** - 可延长至 48 小时

### 输出格式

```

【共识点】
- ...

【分歧点】
- ...

【独特视角】
- ChatGPT: ...
- Gemini: ...

【综合得分】X.X/1.0

【Nia 建议】...

【待主任决策】...
```
