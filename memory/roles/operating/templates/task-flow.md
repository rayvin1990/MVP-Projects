# 工作流程规范 - 任务分配

**版本:** v1.0  
**创建时间:** 2026-03-15  
**CEO:** nia

---

## 核心原则

**任务必须放到共享记忆区！**

所有任务分配必须放到以下路径：
```
memory/roles/operating/tasks/
```

---

## 任务分配流程

1. **CEO 任命 PM** → 指定项目负责人
2. **PM 分配任务** → 创建任务文件到 `memory/roles/operating/tasks/`
3. **Agent 读取任务** → 从共享记忆区获取任务
4. **执行并汇报** → 完成时报送到共享记忆区

---

## 文件命名规范

```
{agent类型}-{项目名}.md
```

示例：
- `codex-zhuangmei.md` - Codex 的妆妹任务
- `claude-code-zhuangmei.md` - Claude Code 的妆妹任务

---

## 错误记录

### 2026-03-15 错误
- **错误：** 任务放到了项目文件夹而不是共享记忆区
- **后果：** Codex 看不到任务，项目停滞
- **教训：** 任务必须放到共享记忆区
